Tipos de Datos

  1. int númerico entero. Ej: a = 50
  2. float númerico decimal. Ej: b = 12.12
  3. str cadena de caracteres. Ej: c = "¡hola, mundo! (mayo, 2025)"
  4. bool valores booleanos. Ej: d = True
  5. list lista de datos. Ej: frutas = ["toronja", "limon", "piña"]
  6. tuple tupla de datos Ej: coordena = (12,7)
  7. dict Diccionario clave, dato. Ej: mascote = {"especie": "perro", "raza": "golden", "nombre": "solovino"}
