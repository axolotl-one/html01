Declaraciones de Variables

Sintaxis para Declaración de Variables

tipo_de_dato identificador = valor

Errores de Sintaxis para Declaración de Variables

01. int valor#1 = 0;
  > Error al usar caracteres especiales "#, &, %, !, ñ, ..."
02. int tamaño = 5;
  > Error al usar caracteres especiales "#, &, %, !, ñ, ..."
03. Int x = 87;
  > Error al usar el declarador de tipo con mayuscula
04. int y = 30
  > Error al no terminar la declaración con punto y coma ";"
05. int z = 7 < 10;
  > Error al asignar un dato de tipo incompatible (asigna un dato bool en una variable int)
06. double 8vo = 1 / 8; 
  > Error al iniciar el ID con un número"
07. float decimal = 4;
  > Error al usar una palabra reservada como ID"
08. float n = 8.23
  > Error al no colocar el sufijo "f" para float después del valor númerico
09. byte valor = 300;
  > Error al usar un valor que excede el limite de bits del tipo de dato
10. int z = 7;
  > Error al declarar una variable con ID ya existente (linea 5)
11. char letra = "a";
  > Error al usar comillas dobles para char, se debe usar comillas simples

Operador +

1. Identifica el tipo de dato de ambos operandos

2. Si ambos son de tipo númerico, el operador + realiza una suma aritmetica.

3. Si al menos uno de los operandos es string, el operador + lo convierte a string y los une en un solo string

 30 + 7     // int + int = int => 30 + 7 = 37
"30" + 7   // str + int => str + str => "30" + "7" = "307"
 7 + "30"   // int + str => str + str => "7" + "30" = "730"
 7 + 3 + "5"  /*
int + int + str
 7  +  3
   10   +  "5"
   int  +  str
   str  +  str
  "10"  +  "5"
  = "105"

Errores de Concatenación

int + bool => error por incompatibilidad de los tipos de datos
int + int => error si el resultado se asigna a un string

Vectores (Arreglos Unidimensionales)

Sintaxis de Vector Vacío de N-Elementos

tipo_de_dato[]  nombre_vector  = new tipo_de_dato[n_elementos];
// Genera elementos vacíos: { 0, 0, 0, ..., n_elementos }

Sintaxis de Vector Precargado en el Programa

tipo_de_dato[] nombre_vector = { elemento01, elemento02, elemento03, ..., elementoN }
// Genera elementos del tipo de dato declarado

Matrices (Arreglos Bidimensionales)

Sintaxis de Matriz Vacía de N-Elemntos

tipo_de_dato[,] nombre_matriz = new tipo_de_dato[m_elementos, n_elementos]
// genera elementos vacíos: { 0, 0, 0, ..., m*n_elementos }

Sintaxis de Matriz Precargada en el Programa

tipo_de_dato[,] nombre_matriz =
{
   { e11, e12, e13, ..., e1n },    // fila 0
   { e21, e22, e23, ..., e2n },    // fila 1
   { e31, e32, e33, ..., e3n },    // fila 2
   { ..., ..., ..., ..., ... },    // ...
   { en1, en2, en3, ..., enn }     // fila n
}
// genera la matriz con elementos del tipo de dato declarado

Longitudes

matriz.length // Devuelve la cantidad de elementos totales de la matriz

matriz.GetLength(0) // Devuelve el tamaño de filas

matriz.GetLength(1) // Devuelve el tamaño de columnas

Matrices 3D

Sintaxis de Matriz Vacía de N-Elemntos

tipo_de_dato[,,] nombre_matriz = new tipo_de_dato[x_elementos, y_elementos, z_elementos]
// genera elementos vacíos: { 0, 0, 0, ..., x*y*z_elementos }

Sintaxis de Matriz Precargada en el Programa

tipo_de_dato[,] nombre_matriz =
{
   {  // Matriz 1
      { e111, e112, e113, ..., e11n },    // fila 0
      { e121, e122, e123, ..., e12n },    // fila 1
      { e131, e132, e133, ..., e13n },    // fila 2
      { ..., ..., ..., ..., ... },    // ...
      { e1n1, e1n2, e1n3, ..., e1nn }     // fila n
   },
   {  // Matriz 2
      { e211, e212, e213, ..., e21n },    // fila 0
      { e221, e222, e223, ..., e22n },    // fila 1
      { e231, e232, e233, ..., e23n },    // fila 2
      { ..., ..., ..., ..., ... },    // ...
      { e2n1, e2n2, e2n3, ..., e2nn }     // fila n
   },
   // ...
   {  // Matriz N
      { en11, en12, en13, ..., en1n },    // fila 0
      { en21, en22, en23, ..., en2n },    // fila 1
      { en31, en32, en33, ..., en3n },    // fila 2
      { ..., ..., ..., ..., ... },    // ...
      { enn1, enn2, enn3, ..., ennn }     // fila n
   },
}
// genera un arreglo de matrices con elementos del tipo de dato declarado
// también puede interpretarse como una matriz cubica o como un paralelepípedo
