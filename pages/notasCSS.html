Conceptos Básicos

CSS ("Cascading Style Sheets" u "Hoja de Estilos en Cascada")

Es un lenguaje utilizado para darle estilo a una página web, un maquillador que la hace atractiva para los usuarios.

Separa el contenido de una página web de su diseño, organiza mejor el código para el programador y hace la página más ágil

  1. Aplicar estilos de manera selectiva
  2. Crear diseños reponsivos
  3. Adaptar la página a multiples tamaños de pantalla

Vinculación de una archivo CSS en HTML:

<link rel="stylesheet" href="style.css">

Selectores

Ejemplo de Estructura de un Selector

p { 
   color : red; /* Cambia el color de fuente a rojo */
}

p: es el selector (id, clase, tipo, étc)

color: es una propiedad del selector

red: es un valor de la propiedad mencionada

color : red;: juntas hacen una declaración para el selector

/* Cambia el color de fuente a rojo */: es un comentario

Modelo de Cajas (Box Model)

            

                
box-sizing

                

                    
ValoresDescripción

                    
content-box(valor predeterminado) al definir el alto o largo de un elemento, se definirá solo sobre el contenido

                    
border-boxAl definir el largo o alto de un elemento, se definirá sobre el borde y se incluirá el borde, padding y contenido

                

            

            

                
margin

                
Define las dimensiones del margen externo (fuera del borde) a partir de un valor numerico junto con una proporción (px, %, vh, vw, em, rem...)

                

                    
ValoresDescripción

                    
valor-1valor-1 asigna el valor longitudinal para margin-top, margin-left, margin-right, margin-bottom;

                    
valor-1 valor-2
valor-1 asigna el valor longitudinal para margin-top, margin-bottom;

                        
valor-2 asigna el valor longitudinal para margin-left; margin-right

                    
valor-1 valor-2 valor-3
valor-1 asigna el valor longitudinal para margin-top;

                        
valor-2 asigna el valor longitudinal para margin-left; margin-right;

                        
valor-3 asigna el valor longitudinal para bottom

                    
valor-1 valor-2 valor-3 valor-4

Propiedades Posicionales

            

                
display

                

                    
ValoresDescripción

                    
inlineOrdena al elemento mismo en una misma linea horizontal junto con otros elementos inline, útil para textos.

                    
inline-blockOrdena al elemento mismo en una misma linea con compartamiento interno block, útil para barras de navegación.

                    
blockOrdena al elemento mismo como elemento separado verticalmente, útil para secciones básicas.

                    
flexPermite manipular el orden y posición de los elementos hijos con respecto a un eje unidimensional.

                    
gridPermite manipular el orden y posición de los elementos hijos con respecto a un sistema de coordenadas.

                    
noneIgnora por completo la existencia del contenedor

                

            

            

                
position

                

                    
ValoresDescripción

                    
static(valor predeterminado) Inhabilita el movimiento de top, bottom, left y right;

                    
relativeRealiza cualquier desplazamiento adicional a partir de su posición inicial, ocupa el espacio original

                    
absoluteRealiza cualquier desplazamiento adicional a partir del punto (0,0) del contenedor padre

                    
fixedSu posición permanece fija con respecto a pantalla del dispositivo

                    
stickySu posición permanece fija con respecto a su contenedor padre y estática fuera de él

                

            

            

                
z-index

                
Evalúa el nivel de capas de superposición en el que apareceran los elementos, un valor entero positivo aparecerá sobre otro de mivel menor o negativo.

            

        

        

            
Propiedades de display: flex;

            

                
flex-direction

                

                    
ValoresDescripción

                    
row(valor predeterminado) Ordena los elementos sobre el eje horizontal de izquierda a derecha

                    
row-reverseOrdena los elementos sobre el eje horizontal de derecha a izquierda

                    
columnOrdena los elementos sobre el eje vertical (Y) desde la parte superior a la parte inferior

                    
column-reverseOrdena los elementos sobre el eje vertical (Y) desde la parte inferior a la parte superior

                

            

            

                
justify-content

                
Alinea o justifica el contenido del contenedor padre con respecto al eje principal

                

                    
ValoresDescripción

                    
flex-start(valor predeterminado) Alinea al comienzo sin justificar el contenido

                    
flex-endAlinea al final sin justificar el contenido

                    
centerCentraliza los elementos sin justificar el contenido

                    
space-betweenJustifica utilizando todo el largo con espacios equitativos entre los elementos, pero sin espacio entre los bordes laterales

                    
space-aroundJustifica utilizando todo el largo con espacios equitativos entre los elementos, pero dejando la mitad de espacio en los bordes laterales

                    
space-evenlyJustifica utilizando todo el largo con espacios equitativos entre los elementos sin distinción con los bordes laterales
