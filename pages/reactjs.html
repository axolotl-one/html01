Hooks

  1. useRef

    Referencia al DOM (elementos del DOM), este hook devuelve un objeto mutable con la propiedad .current;

    Persiste durante todo el ciclo de vida del componente

    Actualizar su valor no provoca un nuevo renderizadao

    Es como una "caja" donde se guarda algo sin que React se enetere de sus cambios

    Ejemplos de uso: manejar el foco de un input, el ID de un temporizador

    Manejo del foco de un input

    import { useRef } from react;
    function FocusInput() {
         // const refObject = useRef( valorInicial )
         const inputApuntado = useRef(null);
         const onButtonClick = () => {
              inputApuntado.current.focus();
              // current apunta al elemento de input montado
         }
         return (
              <>
                   <input ref = {inputApuntado} type = "text"/>
                   <button onClick = {onButtonClick}>Poner foco en el Input</button>
              </>
         )
    }

  2. useCallback

    Memorizar la definición de una función para que no se cree una nueva instancia de esa función en cada renderizado

    En JS, una función es un objeto

    Estructura

    import { useCallback } from react;
    function principal() {
         useCallback(
              () => {
                   function(a, b)
              },
              [a,b]
         );
    }

  3. useReducer

    Maneja lógica de estado compleja o interdependiente

    Centraliza lógicas de actualización en una función reductora

    Ejemplo de aplicación: actualización de un contador

    import { useReducer } from react;
    // Se recomienda aplicar las funciones reductoras fuera de la función del componente para reutilizar en otros archivos
    // Si se declara la función reductora dentro del componente, únicamente aplicará a ese componente
    const estadoInicial = { contador : 0 }
    function reducer(state, action) {
         switch (action.type) {
              case 'incrementar':
                   return { contador : estado.contador + 1; }
              case 'decrementar':
                   return { contador : estado.contador - 1; }
              case 'resetear':
                   return { contador : 0; }
              default:
                   throw new Error();
         }
    }
    function Contador() {
         const [state, dispatch] = useReducer(reducer, estadoInicial)
         return (
              <>
                   Contador: { state.contador }
                   <button onClick = {() => dispatch({ type: 'incrementar' })}>+</button>
                   <button onClick = {() => dispatch({ type: 'decrementar' })}>-</button>
                   <button onClick = {() => dispatch({ type: 'resetear' })}>Resetear</button>
             </>
         );
    }
    export default Contador;

    Ejemplo de aplicación: Actualización de un Temporizador

    import { useRef } from 'react';
    function Temporizador() {
         const timerRef = useRef(null);
         const iniciarTemporizador = () => {
              timerRef.current = setTimeout(() => {
                   alert('Tiempo agotado!');
              }, 3000);
         };
         const cancelarTemporizador = () => {
              clearTimeout(timerRef.current);
         };
         return (
              <div>
                   <button onClick={iniciarTemporizador}>Iniciar</button>
                   <button onClick={cancelarTemporizador}>Cancelar</button>
              </div>
         );
    }

react-router-dom

Se encarga de gestionar la barra de búsqueda del navegador para simular rutas URL

Componentes

<BrowserRouter>{/*...*/}</BrowserRouter>

- Envuelve a la raíz de la aplicación para que router-dom sea ejecutable.

- Conecta a la barra de búsqueda del navegador para gestionar el historial

<Routes>{*...*}</Routes>

- Envuelve una lista de rutas y renderiza a la primera ruta que coincida con la URL actual

<Route path='url' element={<Componente />} />

- Asigna una URL a un componente

<Link to='url'>Click Aquí</Link>

- Sustituye a la etiqueta <a>

npx create-vite@latest

Estructura de Emboltura del Proyecto

import React from "react";
import ReactDOM from "react-dom/client";
import { BrowserRouter } from "react-router";
import App from "./app";
 
const root = document.getElementById("root");
 
ReactDOM.createRoot(root).render(
  <BrowserRouter>
    <App />
  </BrowserRouter>,
);

Estructura de Enrutamiento

import { BrowserRouter, Routes, Route } from 'react-router-dom'
import Inicio from 'rutaLocalComponente1'
import ComponenteDeRuta2 from 'rutaLocalComponente2'
import ComponenteDeRuta3 from 'rutaLocalComponente3'
// ...
 
function App() {
     return (
          <BrowserRouter>
               <Routes>
                    <Route path='/' element={<Inicio />} />
                    <Route path='/urlRuta2' element={<ComponenteDeRuta2 />} />
                    <Route path='/urlRuta3' element={<ComponenteDeRuta3 />} />
               </Routes>
          </BrowserRouter>
     );
};
 
export default App

Ejemplo de Hook useParams

En el componente padre:

import componenteHijo from './rutaLocal'
<Routes>
     {/*Otras rutas*/}
     <Route path="/url/ :id" element={<componenteHijo />} />
</Routes>

En el componente hijo:

import { useParams } from 'react-router-dom'
function componenteHijo() {
     const { id } = useParams();
     return(
          <div>
               <p>Detalles del objeto:</p>
               <p>ID: { id }</p>
          </div>
     )
}
