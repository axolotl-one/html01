Notas de Comandos

Terminal

    Nombre del Comando Descripción
  • cls
    		• Limpiar pantalla
  • ls
    		• Mostrar elementos contenidos
  • mkdir
    		• Crear nueva carpeta
  • cd ..
    		• Retroceder un nivel
  • cd nombre
    		• Accede al elemento del nivel por nombre ingresado
  • q
    		• Salir de un archivo cuando arroja (END)

Git

    Nombre del ComandoDescripción
  1. git status
    		2. Muestra el estado del commit (sin repo, libre, con cambios pendientes)
  2. git init
    		3. Crea un nuevo repositorio
  3. git add -A
    		4. Anexa todos los cambios pendientes (nuevo, modificado o eliminado)
  4. git commit -m "etiqueta"
    		5. Crea un nuevo commit (version) en la rama actual, etiqueta forzosa
  5. git log
    		6. Enlista todos los commits de la linea temporal (id, etiqueta, usuario, fecha)
  6. git log --oneline
    		7. Enlista todos los commits de la linea temporal (hash, etiqueta)
  7. git checkout hash
    		8. Visualiza la version del hash ingresado, util para revisiones
  8. git checkout NombreRama
    		9. Direcciona la rama
  9. git checkout -b NombreRama
    		10. Direcciona la rama, si no existe, la crea
  10. git switch --detach
    		11. Hash direcciona el puntero HEAD a la version con el hash ingresado
  11. git switch -c Nombre
    		12. Crea una nueva rama a partir de la version a la que apunta el HEAD
  12. git branch
    		13. Muestra la lista de ramales
  13. git branch -D NombreRama
    		14. Elimina la rama con el nombre ingresado
  14. git cherry-pick hash
    		15. Anexa en la version actual (HEAD) los cambios de la version del hash
    En conflicto por sobreestricuta se puede: Conservar cambios de la version pasada, de la actual o ambos
  15. git merge
    		16. Anexa en la version actual los cambios de un ramal y extrae su historial de versiones
    En conflicto por sobreestricuta se puede: Conservar cambios de la version pasada, de la actual o ambos
  16. git clone https://link-github
    		17. Clona un repositorio remoto al local en una nueva carpeta del mismo nombre
  17. git remote add aliasRepo https://link-github
    		18. Anexa un repositorio remoto de github a git y lo nombra
  18. git remote get-url aliasRepo
    		19. Obtiene la url del repositorio ingresando su nombre
  19. git push aliasRepo master
    		20. Envia y sube las versiones de la rama master al repositorio remoto
  20. git push aliasRepo ramaLocal:ramaRemota
    		21. Enivia las versiones de la rama local a una remota nueva o existente
  21. git pull aliasRepo
    		22. Actualiza el repositorio local con los cambios de otros colaboradores
  22. git fetch aliasRepo
    		23. Descarga los cambios de otros colaboradores para revisarlos antes de aplicarlos

Vim

  • esc + :q
    		• Salir solo si no hay cambios pendientes
  • esc + :q!
    		• Salir sin guardar cambios
  • esc + :wq
    		• Guardar y salir
