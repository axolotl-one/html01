Notas de Comandos

Terminal

  • cls limpiar pantalla
  • ls mostrar elementos contenidos
  • mkdir crear nueva carpeta
  • cd .. retroceder un nivel
  • cd name accede a un elemento del nivel

Git

  • status muestra el estado del commit (sin repo, libre, con cambios pendientes)
  • init crea un nuevo repositorio
  • add -A anexa todos los cambios pendientes (nuevo, modificado o eliminado)
  • commit -m crea un nuevo commit (version) en la rama actual
  • log enlista todos los commits de la linea temporal (id, etiqueta, usuario, fecha)
  • log -oneline enlista todos los commits de la linea temporal (hash, etiqueta)
  • checkout hash visualiza la version del hash ingresado, util para revisiones
  • checkout rama direcciona la rama
  • switch --detach hash direcciona el puntero HEAD a la version con el hash ingresado
  • switch -c name crea una nueva rama a partir de la version a la que apunta el HEAD
  • branch muestra la lista de ramales
  • remote add name https://link... anexa un repositorio remoto a github
  • push name master envia las versiones de la rama master al repositorio remoto
